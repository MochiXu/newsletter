## 1. 后端对外契约加厚(`briefs.json`)

V1 的 `briefs.json` 只有四层 + 指标表。这一阶段把"代码算好的东西"尽量暴露给前端,并让假设层变成可评估的预测。

| 增强 | 内容 | 关键代码 |
|---|---|---|
| **technical signals** | 29 条技术指标(趋势/动量/波动/利率·通胀/美元/跨资产相关/52周分位),带 `unit`(pct/pct0/bp/z/corr/yield)+ `group`;单一事实源 `features.FEATURE_VIEW` | `render.build_signals` |
| **regime 标签** | 6 个代码派生的市场状态(股票趋势/波动/曲线/实际利率/通胀预期/美元) | `regime.derive` |
| **假设层 → 预测卡** | 假设层改为对**固定 roster**(纳指 NASDAQCOM / 黄金 XAUUSD / 广义美元 DTWEXBGS / 2Y DGS2)各给且只给一条预测,带 `asset/direction/horizon/confidence/key_factors/可度量失效` | `catalog.PREDICTION_TARGETS`、`schema` |
| **metric.spark** | 每个指标最近 ~20 个真实收盘点(因果),供前端小走势线 + hover | `features.metric_spark` |
| **priceSeries** | 5 个图表资产(纳指/黄金/广义美元/US10Y/VIX)各 30 个真实价格点(带日期,因果),供前端可交互 30D 价格大图 | `features.price_series`、`render.build_price_series`、`catalog.CHART_SERIES_IDS` |
| **facts/reads 标签化 + 数字方向** | 事实层/解读层从 `str[]` 升级为 `[{tag,text,figures}]`:`tag` 主题标签(受控词表 `models.FACT_TAGS`)、`figures:[{t,dir}]` 标注正文里**有方向含义的变化数字**(供前端按 up/down/flat 上色)。LLM 在 schema 里填**扁平字符串** `figs='token|dir;...'`,`_parse_figs` 解析成结构化 figures | `models.TaggedItem/Figure`、`schema._FIGS_PROP` |
| **新闻升级** | 优质宏观源(Fed/ECB/MarketWatch/CNBC)+ 过滤噪音/无链接 + 带真实 link;fetch 与 classify 解耦(分类失败不丢新闻) | `news.py`、`pipeline` |

**通用文本规范化**(适用所有展示文本:headline / 事实 / 解读 / 影响 / 复盘 / 预测卡):
- `textnorm.normalize_text`(确定性后处理,render 落库前统一应用):全角标点 → ASCII、中文与字母数字之间盘古空格、双引号 → 单引号(避免破坏 JSON);**不拆开** `MA200 / 2s10s / 9bp / 0.27%` 等字母数字词。
- `llm/style.TEXT_STYLE`(公共 prompt 片段,简报 + 新闻共用):向 LLM 表达同样的标点/空格/引号约定,减少后处理改动量。

## 2. LLM 健壮性

| 问题 | 修复 |
|---|---|
| **DeepSeek 强制 function-calling 在复杂 schema 下结构失稳**:加了嵌套字段后,会在 facts 数组后提前闭合根对象、把 interpretation 当兄弟对象续写 → JSON 必解析失败、丢字段 | 结构化输出改用 **JSON mode**(`response_format=json_object` + 把 JSON Schema 写进 system,需含 'JSON' 字样),读 `message.content`;`max_tokens` 4096→8192 |
| figures 用"数组套对象再套数组"的深层嵌套也会触发上面的失稳 | figures 在 schema 里用**扁平字符串** `figs='token|dir;...'`,后端解析(扁平串 LLM 友好,前端契约仍是 `figures:[{t,dir}]`) |
| 新闻分类失败会把已抓到的新闻一并丢掉 | fetch / classify 解耦,分类失败降级为"未分类新闻"仍展示 |

## 4. 当前 `briefs.json` 契约(完整)

顶层 `{model, generatedAt, briefs:[Brief]}`。`Brief`:
`date / weekday / issue / time / tone / headline`
· `metrics[{key,label,value,change,kind,spark:number[]}]`
· `signals[{key,label,value,unit,group}]` · `regime{}`
· `priceSeries{key:[{date,value}]}`(5 资产×30 点)
· `facts[{tag,text,figures:[{t,dir}]}]` · `reads[同]`
· `hypotheses[{ifThen,invalidation,asset,direction,horizon,confidence,keyFactors}]`
· `impacts[{asset,watch,dir}]` · `reviews[{ifThen,status,note}]`
· `news[{title,source,cat,assets,dir,link}]`
单一事实源 = `py/newsletter/models.py`;所有展示文本经 `textnorm.normalize_text`。
