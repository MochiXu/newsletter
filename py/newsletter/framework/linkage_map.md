# 宏观传导图(linkage map)· v1

> 本文件是 newsletter 的核心 IP:一份**人工维护**的宏观因果/传导关系清单。
> AI 据此推理,生成每日简报的解读层与假设层;你每天复盘后修订它。
> 它是规则与先验,不是铁律——市场是 regime-dependent 的,尽量标注「在什么环境下成立」。

## 利率 / 收益率曲线
- 10Y 收益率 ↑:压制长久期资产(成长股、黄金、长债);**实际利率**(10Y − 盈亏平衡通胀)比名义更关键。
- 2Y 收益率:更贴近市场对美联储政策路径的定价;2Y 急升 = 加息 / higher-for-longer 预期升温。
- 2s10s(10Y − 2Y):
  - 走平 / 倒挂:衰退担忧、紧缩预期。
  - 深度倒挂后**转陡**(尤其牛陡:2Y 快速下行)常预示降息临近 / 衰退确认。
  - 熊陡(10Y 比 2Y 涨更快):再通胀 / 期限溢价上升 / 财政担忧。

## 美元
- 广义美元(DTWEXBGS)与 DXY(DX-Y.NYB)方向通常一致,但 DXY 偏欧日英、广义偏贸易伙伴;两者背离本身是信号。
- 美元 ↑:压制大宗、黄金、新兴市场、美股海外营收;risk-off 时美元常作避险买盘。
- 美元 ↓ + 实际利率 ↓:对黄金、新兴市场、大宗最友好的组合。

## 波动率 / 风险偏好
- VIX < 15:低波动、risk-on;15–20:中性偏震荡;> 20 且上行:risk-off,关注信用利差。
- VIX 急升常伴随股跌、美元/美债避险买盘、流动性收紧。

## 黄金
- 主要逆风:实际利率 ↑、美元 ↑。
- 主要顺风:实际利率 ↓、美元 ↓、避险需求、央行购金、通胀预期升温。
- 与实际利率长期负相关,但在危机 / 去美元化叙事下可能脱钩。

## 跨资产组合信号(示例)
- 「美元 ↑ + 实际利率 ↑ + VIX ↑」= 典型 risk-off / 紧缩冲击:股、金、新兴普遍承压。
- 「2Y ↓ + 2s10s 走陡 + 美元 ↓」= 降息预期主导:利好成长股、黄金、长债。
- 数据彼此矛盾时(如美元 ↑ 但黄金也 ↑)= 可能有避险 / 结构性买盘,值得在简报里点名为「背离观察点」。

## 维护说明
- 每条尽量写明「在什么 regime 下成立 / 何时失效」。
- 新学到的关系 → 加进来;被证伪的 → 标注或删除。
- 这是学习载体:简报里的假设第二天复盘,对了/错了都回头修订这张图。
