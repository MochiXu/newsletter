# 市场走势简报 · 前端界面设计文档

> 一份由 AI 生成的每日金融宏观简报阅读器。本文档用文字描述该前端界面的设计理念、信息架构、视觉系统与交互模式,供设计/产品/工程参考。

---

## 1. 设计目标

- **承载形式**:每个交易日生成一份独立的「小票式」简报卡片。
- **信息密度高但不杂乱**:同时容纳宏观指标数值、走势、四层 AI 简报、新闻分类、假设复盘等内容。
- **可扫读 + 可深读**:用户能在 5 秒内把握当日基调,也能停下来逐项细看。
- **克制的动态效果**:仅在关键交互上使用淡入淡出与高亮,不喧宾夺主。
- **明暗主题一致**:跟随系统主题,同时提供手动切换,所有色彩通过 CSS 变量统一调度。

## 2. 整体气质 / 美学方向

**「暖纸质感小票」**——灵感来自便利店热敏小票:
- 主背景是浅米黄(暖灰),卡片本体是暖白纸色。
- 顶部与底部使用「半圆扇贝」边缘,模拟撕下来的小票边。
- 等宽字体承担所有数据与日期,衬托「打印输出」的可信感。
- 虚线分隔、条形码尾签、星号包夹的水印行,作为视觉点缀但不滥用。
- 深色模式下,背景切换为低饱和的咖啡棕黑,纸色变深褐,所有暖色调保留,避免冰冷的纯黑科技感。

## 3. 信息架构

```
顶部页眉 ─────────────────────────────────────
│ 站标 + 副标题(左)    主题切换(右)
├─────────────────────────────────────────────
│
│  时间线(左栏)        │   小票卡片(主视觉,居中偏右)
│  ─ 9 个交易日按倒序  │   ─ 当日所有内容,垂直阅读
│  ─ 悬停淡入预览      │
│  ─ 点击锁定          │
│
└─────────────────────────────────────────────
```

整体采用 **左右分栏 + 上方页眉** 的三段式布局:
- 页眉:站点身份、定位、主题切换。
- 左栏:时间线导航,占约 1/4 宽度。
- 右栏:小票卡片,占约 3/4 宽度,水平居中。
- 在窄屏时左右栏自动堆叠,小票仍居中显示。

## 4. 顶部页眉

从左到右:
- **栏目标识**:`DAILY MACRO BRIEF` 用等宽字、橙赭色、字间距 2.5px,呈现「电报式」标识。
- **主标题**:「市场走势简报」无衬线粗体,字号约 27px,负字距 -0.5px。
- **副标题**:「AI 生成 · 金融数据抓取 + 走势研判 · 每个交易日一刊」,小字、低饱和。
- **主题切换器**(右侧):一组 segmented 控件,三档:`自动` / `浅色` / `深色`,选中态有轻微阴影与背景反白。
- 页眉与下方内容之间有一根细横线,作为「水位线」。

## 5. 时间线(左栏)

这是用户切换日期的主控件。

- **形态**:一根竖直的细线串起若干圆点,圆点旁是日期与该日标题。
- **颜色编码**:每个圆点的颜色代表当日的市场基调:
  - 绿色 = risk-on(避险情绪松弛)
  - 红色 = risk-off(避险情绪上升)
  - 暖灰 = neutral(中性/观望)
- **未激活态**:文字与圆点轻微降低不透明度,远观就是一串安静的时间锚点。
- **激活态**:
  - 该行背景填充一层更深的暖色卡纸色;
  - 圆点放大,外层多一圈橙色光环;
  - 日期/标题文字回归全饱和。
- **交互**:
  - **悬停**:把右侧小票淡出再淡入到这一天(预览模式,不修改锁定状态)。
  - **离开**:回到当前锁定的那一天。
  - **点击**:锁定到这一天。
- **底部小字**:`悬停预览 · 点击锁定`,作为隐性提示。

## 6. 小票卡片(主视觉)

这是整个产品的核心载体。从上到下分若干模块,以虚线分隔。

### 6.1 撕边 + 抬头

- 顶部与底部分别是一排白色半圆扇贝,模拟小票撕下来的锯齿。
- 抬头分三行:
  - 第一行:`宏观简报 · MACRO BRIEF`(等宽小字,字间距大)。
  - 第二行:**当日日期**,用大号等宽字呈现,如 `2026-06-16`。
  - 第三行:`周二 · 第 142 刊 · 07:00 CST`,等宽小字。
- 整个抬头居中,呼应小票的对称美学。

### 6.2 一句话标题(Headline)

整刊最重要的一句话,粗体、字号略大、宽松行高,概括当日基调,例如:
> 「FOMC 鹰派暂停:点阵图上修,10Y 与美元齐升,VIX 跳上 17。」

### 6.3 当日指标表(MARKET DATA)

- 模块头:一行小标签 `· MARKET DATA  当日指标 -------`,左边一颗小圆点,右边一根虚线。
- 表格固定 6 行,每行四列:
  1. **指标名**:US10Y、US2Y、2s10s、VIX、DXY、GOLD(等宽小字)。
  2. **迷你走势线(sparkline)**:18 点折线,无坐标轴,末端有一颗小圆点表示「现在」。颜色随当日涨跌:绿色=上涨、红色=下跌、灰色=持平。
  3. **数值**:右对齐,等宽,根据指标自动选用单位(`%`、`bp`、`小数`、`千分位`)。
  4. **变化量**:小字,带正负号,沿用涨跌色。
- 每行底部一根虚线,密但不重。

### 6.4 四层 AI 简报(AI BRIEF)

这是这份产品的方法论核心,严格分为四个子层,每层一个小方块图标 + 中英文双标签:

- **■ FACTS 事实层**:以 `›` 开头的几条要点,纯陈述,墨色为主。
- **■ INTERPRETATION 解读层**:以 `—` 开头,小标记为橙赭色,正文略浅,表达「这是模型的解释」。
- **■ HYPOTHESIS 假设层**:每条假设以浅色卡片单独呈现(暖纸第二层色),内含两行:
  - 一行「若 X,则 Y」的可证伪命题,粗一档。
  - 一行 `✕ 失效` + 失效条件(小字、红色标记)。
- **■ IMPACT 影响层**:每条形如 `↑ 资产 ····· 观察点`,箭头表示方向偏多/偏空/待观察,资产名等宽,后接一段灰色的关注点描述。

四层之间用纵向间距分隔,不再额外加分割线,以免视觉太满。

### 6.5 假设复盘(REVIEW)

回看前一天/前几天的假设,是这套方法论的「记账本」。

- 每条复盘左边是一个圆形小徽章:
  - `✓` 绿色 = 已兑现
  - `✕` 红色 = 已失效
  - `○` 橙色 = 待观察
- 右边两行:
  - 第一行:原始的「若 X,则 Y」命题。
  - 第二行:状态标签(小字、对应色) + 复盘备注。
- 如果当日没有可复盘的假设,整个模块隐藏。

### 6.6 新闻分类(NEWS)

把当日相关新闻按角色打标签呈现。

- 每条新闻一行,左侧是一个「分类徽章」:
  - `事实`:墨色描边,深色文字。
  - `解读`:橙赭色描边 + 同色文字。
  - `事实+解读`:蓝色描边 + 同色文字。
  - `噪音`:浅灰描边,正文也降低饱和度,提醒读者「可跳过」。
- 中间是新闻标题(英文为主,保留原始措辞)。
- 右侧是方向箭头(↑/↓/·),沿用涨跌色。
- 下方一行小字:`新闻来源` + 若干「受影响资产」灰色标签(等宽、细描边小方框)。
- 行间用细虚线分隔。

### 6.7 页脚

- 一行小字水印:「本简报仅供研究 · 非投资建议 / NOT INVESTMENT ADVICE」。
- 一段「条形码」:用 CSS 重复线性渐变拼出粗细不一的竖条,无信息含义,纯氛围。
- 最底部:`★ 2026-06-16 · GEN DeepSeek ★`,水平居中,等宽,作为收尾签名。

## 7. 视觉系统

### 7.1 字体

- **等宽字** `IBM Plex Mono`:数据、日期、标签、章节小标题、条形码下的签名。
- **无衬线** `Noto Sans SC`:正文、标题、说明性文字,中文显示更端正。
- 两种字号阶层(粗略):
  - 大标题 27px
  - 小票日期 28px(等宽)
  - 一句话标题 14.5px / weight 600
  - 正文 12–12.5px
  - 标签与水印 9.5–11px

### 7.2 配色(CSS 变量,所有色彩集中管理)

| 角色 | 浅色主题 | 深色主题 | 用途 |
| --- | --- | --- | --- |
| `--bg` | 暖米黄 #e7ddc9 | 咖啡黑 #16130f | 页面背景 |
| `--paper` | 暖白 #fdfaf2 | 深褐 #221c15 | 小票纸面 |
| `--paper2` | 米卡纸 #f3ebd9 | 浅一档深褐 #2b241b | 假设卡、激活行 |
| `--ink` | 墨黑 #2b2018 | 米白 #ece2cf | 正文文字 |
| `--ink2` | 暖灰 #8a785b | 沙褐 #9a8a6f | 弱文字 |
| `--faint` / `--hair` | 浅米线 | 深褐线 | 虚线、描边 |
| `--accent` | 橙赭 #c0612f | 暖橙 #e0824a | 重点小圆点、解读层、橙色徽章 |
| `--up` | 墨绿 #2f7d50 | 亮绿 #5bc486 | 上涨/已兑现 |
| `--down` | 红土 #c0432a | 朱红 #e8694e | 下跌/已失效 |
| `--blue` | 钢蓝 #3a6ea5 | 雾蓝 #7aa6dd | 「事实+解读」分类、假设层标记 |

主题切换通过 `data-theme` 属性 + `prefers-color-scheme` 媒体查询联动,所有颜色一次性切换,有 0.35s 缓动过渡。

### 7.3 形状与节奏

- 圆角:小票本体不用圆角(更像真实的小票);徽章 3–4px;按钮/选项卡 7–9px。
- 投影:小票下方一层暗向阴影,轻微抬起感。
- 间距:模块之间约 20px,模块内行间约 5–10px,呼吸节奏一致。

### 7.4 纹理

- 一层非常浅的「点阵纹理」叠加在纸面上(径向小圆点 5×5 网格,不透明度 0.28),模拟纸张颗粒。
- 可在 Tweaks 面板里一键关闭。

## 8. 动态效果

刻意克制,只有四处:

1. **小票切换淡入**(`receiptIn` 关键帧):当激活的日期变更时,小票从 12px 下方淡入并上浮 0.42s。
2. **时间线行高亮过渡**:背景色与文字饱和度 0.18s 淡入。
3. **主题切换过渡**:背景与文字色 0.35s 缓动。
4. **加载占位**:数据尚未就绪时,小票位置是一块带 `mbpulse` 呼吸动画的暖白占位块。

不使用滑入滑出、抖动、弹跳、连续滚动跟随等装饰性动效。

## 9. 用户可调项(Tweaks)

通过宿主右侧面板提供 3 个开箱即用的调节:

- **主题色 `accent`**:色板选择橙赭/暖橙等,实时刷新所有重点高亮。
- **小巧走势线 `showSparklines`**:开/关,关闭后指标表只剩名称+数值,更纯粹。
- **纸张纹理 `paperTexture`**:开/关,关闭后小票纸面是纯色。

## 10. 状态与边界情况

- **加载中**:小票占位为暖色块 + 呼吸动画,左栏时间线为空。
- **某日无复盘**:`REVIEW` 模块整段隐藏,不留空标题。
- **新闻为空**:`NEWS` 模块只剩章节头(实际数据生成保证至少 2 条)。
- **主题偏好持久化**:用户手动选择的主题写入 `localStorage`,下次自动恢复。
- **窄屏**:左右两栏自动垂直堆叠,小票仍然水平居中并保持最大宽度 442px。

## 11. 后续可拓展方向

- **详情页 / 第二屏**:点击小票上的某一指标行,展开为带坐标轴的完整走势图(可对比多指标)。
- **跨日搜索**:按资产或关键词检索所有历史小票。
- **「小票打印」导出**:把当日小票导出为 PDF / PNG,直接打印出来,贴在交易笔记本上。
- **多模型对比**:同一天同时展示 DeepSeek / GPT / Claude 三份小票,横向比较解读差异。
- **失效自动标注**:模型在 N 天后自动回标历史假设的兑现/失效状态,持续训练这个「假设账本」。
